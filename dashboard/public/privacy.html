Privacy notice

Short version: this site collects no personal data.

What we collect

  • No accounts, no cookies, no analytics, no tracking. The dashboard is a static site serving public benchmark data (model names, task ids, test outcomes, run metrics) — none of it personal.
  • Standard, transient web-server access logs may be processed by the static host (Cloudflare Pages) for security and delivery, per that provider's policy.
  • The optional live playground runs only pre-vetted tasks, stores no personal data, accepts no free-form input to the model, and is rate-limited.

Third parties

Per-run traces link to Langfuse (observability) for transparency; following such a link opens that third party's service under its own policy. Source code and CI run on GitHub.
